# CPI Examples for DAMM v1

This repository contains examples for Cross-Program Invocation (CPI) to swap using Dynamic AMM programs.

**Repository:** [https://github.com/MeteoraAg/cpi-examples](https://github.com/MeteoraAg/cpi-examples)

## Dependencies

- **Anchor:** `0.28.0`
- **Solana:** `1.16.1`
- **Rust:** `1.68.0`

## Available Examples

The repository includes the following CPI examples:

- **CPI to Dynamic AMM swap example**
- **CPI to Dynamic AMM initialize pool example**
- **CPI to Dynamic AMM initialize pool with config example**
- **CPI to Dynamic AMM lock liquidity example**
- **CPI to Dynamic AMM claim fee example**
- **CPI to M3m3 initialize vault example**

## Testing

The repository includes comprehensive tests for all the CPI examples.

## Additional Resources

For more detailed information, please refer to the Dynamic AMM repository.
