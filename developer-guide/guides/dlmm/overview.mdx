# DLMM Program

At Meteora, we’ve developed a `Node.js <> Typescript SDK` and a `Rust CPI Examples` to make deploying and managing your DLMM liquidity pool easier. The following sections includes information on installing and using the SDKs and Rust CPIs. It also covers where to find the latest code, and how to contribute to these repositories.

We also outline a list of unofficial community SDKs made by our wonderful community.
