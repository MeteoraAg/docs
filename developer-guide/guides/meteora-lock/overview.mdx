# Meteora Lock Program

At Meteora, we’ve developed a `Node.js <> Typescript SDK` to make deploying and managing your Meteora Lock easier. The following sections includes information on installing and using the SDK. It also covers where to find the latest code, and how to contribute to these repositories.
