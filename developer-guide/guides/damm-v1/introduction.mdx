Initialize your own constant product liquidity pool with DAMM v1. DAMM v1 allows you to earn fees from both swap fees and yield rewards from lending platforms.

# Features

<CardGroup cols={2}>
  <Card title="Constant Product Curve" icon="arrows-to-circle" iconType="solid">
    Traditional Uniswap-style `x * y = k` model for general token pairs
  </Card>
  <Card title="Stable Swap Curve" icon="chart-line" iconType="solid">
    Curve-style stable swap with customizable amplification coefficient (`A`) for stable/correlated assets, providing tighter spreads around 1:1 ratio
  </Card>
</CardGroup>

<CardGroup cols={3}>
  <Card title="Permissioned Pools" icon="lock" iconType="solid">
    Admin-controlled pools with custom parameters
  </Card>
  <Card title="Permissionless Pools" icon="globe" iconType="solid">
    Anyone can create pools with default or custom fee tiers
  </Card>
  <Card title="Customizable Pools" icon="wrench" iconType="solid">
    Permissionless pools with flexible parameters
  </Card>
</CardGroup>

<CardGroup cols={2}>
  <Card title="Add Balanced Liquidity" icon="plus" iconType="solid">
    Deposit tokens in pool ratio
  </Card>
  <Card title="Remove Balanced Liquidity" icon="minus" iconType="solid">
    Withdraw tokens proportionally
  </Card>
  <Card title="Add Imbalanced Liquidity" icon="scale-balanced" iconType="solid">
    Deposit in custom ratios (stable pools only)
  </Card>
  <Card title="Remove Single-Side Liquidity" icon="arrow-right" iconType="solid">
    Withdraw only one token type (stable pools only)
  </Card>
</CardGroup>

<CardGroup cols={2}>
