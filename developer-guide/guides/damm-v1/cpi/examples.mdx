This [repository](https://github.com/MeteoraAg/cpi-examples) contains examples for Cross-Program Invocation (CPI) that interacts with the DAMM v1 program.

# Dependencies

- **Anchor:** `0.28.0`
- **Solana:** `1.16.1`
- **Rust:** `1.68.0`
