## Pools

<CardGroup>
  <Card title="get_pools" icon="file-code" iconType="solid" href="/api-reference/damm-v2/pools/get_pools">
    Returns a paginated list of pools using `GET` `/pools`
  </Card>
  <Card title="get_groups" icon="file-code" iconType="solid" href="/api-reference/damm-v2/pools/get_groups">
    Returns a paginated list of pool groups using `GET` `/pools/groups`
  </Card>
  <Card title="get_group" icon="file-code" iconType="solid" href="/api-reference/damm-v2/pools/get_group">
    Returns pools that belong to a specific pool group using `GET` `/pools/groups/{lexical_order_mints}`
  </Card>
  <Card title="get_pool" icon="file-code" iconType="solid" href="/api-reference/damm-v2/pools/get_pool">
    Returns metadata and current state for a single pool using `GET` `/pools/{address}`
  </Card>
  <Card title="get_ohlcv" icon="file-code" iconType="solid" href="/api-reference/damm-v2/pools/get_ohlcv">
    Returns OHLCV candles for a single pool over a time range using `GET` `/pools/{address}/ohlcv`
  </Card>
  <Card title="get_volume_history" icon="file-code" iconType="solid" href="/api-reference/damm-v2/pools/get_volume_history">
    Returns historical volume for a pool aggregated into time buckets using `GET` `/pools/{address}/volume/history`
  </Card>
</CardGroup>
