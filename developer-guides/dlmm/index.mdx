## Resources

Use the public [`MeteoraAg/dlmm-sdk`](https://github.com/MeteoraAg/dlmm-sdk) repository for integration references.

| Path | Use |
| --- | --- |
| `ts-client` | Official `@meteora-ag/dlmm` TypeScript SDK, examples, tests, and scripts. |
| `ts-client/src/dlmm/index.ts` | Main `DLMM` class with pool loading, quotes, transaction builders, positions, rewards, limit orders, and pool creation. |
| `ts-client/src/dlmm/helpers` | PDA derivation, bin arrays, account filters, Token-2022 helpers, quote preparation, and transaction utilities. |
| `ts-client/src/examples` | Small examples for swaps, oracle reads, lock info, bin array initialization, and position flows. |
| `ts-client/scripts` | Runnable devnet scripts for swaps, v2 pool creation, and limit order management. |
| `commons` | Rust quote helpers, PDA helpers, account filters, Token-2022 helpers, and shared types used by repository tooling. |
| `cli` | Command-line utility for managing DLMM workflows. |
| `command_list` | Operator-oriented command examples for fee claims, ILM flows, and bootstrapping status updates. |

## Commands 

| Task | Command |
| --- | --- |
| Build SDK | `cd ts-client && pnpm install && pnpm run build` |
| Run SDK unit tests | `cd ts-client && pnpm run unit-test` |
| Run commons quote tests | `cargo t -p commons --test '*'` |
| Inspect CLI commands | `cargo build && target/debug/cli --help` |

Some end-to-end SDK tests may depend on local validator fixtures or program builds that are not part of the public `lb_clmm` source release. Prefer devnet scripts and transaction simulation for public integration testing.
