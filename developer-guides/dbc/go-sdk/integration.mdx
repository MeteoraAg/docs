The `dbc-go` SDK provides Go helpers for backend services, indexers, and scripts that need to read DBC state or build common DBC transactions.

## Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/MeteoraAg/dbc-go.git
cd dbc-go
go mod tidy
```

## Repository Map

| Path | Use |
| --- | --- |
| `instructions` | Transaction builders for partner, pool, creator, and state-related flows. |
| `math` | Curve and safe math helpers. |
| `common` | Shared constants and types. |
| `helpers` | Account helpers and common utilities. |
| `examples` | Runnable examples for pool creation, swaps, reads, fee claims, and creator transfer. |

## Run Examples

Before running examples, configure the keypairs, RPC endpoint, and public keys required by the file.

```bash
go run examples/<file-name>.go
```

## Notes

Use the TypeScript SDK as the most complete transaction-building reference, especially for newer migration and fee features. Use `dbc-go` when your backend is already Go-based and you need direct control over RPC calls, state reads, or server-side transaction construction.
