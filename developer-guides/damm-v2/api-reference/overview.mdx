The DAMM v2 API is an indexed data API for discovery and analytics. Use it for pool lists, pool groups, pool metadata, OHLCV candles, historical volume, and protocol-level stats.

<Tip>
  The DAMM v2 API is documented with OpenAPI below. Public guidance notes a rate limit of `10` requests per second.
</Tip>

## Base URLs

| Environment | Base URL | Swagger UI |
| --- | --- | --- |
| Production | `https://damm-v2.datapi.meteora.ag` | [Open production Swagger](https://damm-v2.datapi.meteora.ag/swagger-ui/) |
| Development | `https://damm-v2-api.dev.metdev.io` | [Open dev Swagger](https://damm-v2-api.dev.metdev.io/swagger-ui/) |

This docs site uses one OpenAPI file with multiple servers because production and development expose the same API surface.

## API Or On-Chain State

| Need | Use |
| --- | --- |
| Pool discovery, charts, search, dashboards | DAMM v2 API |
| Swap quotes, slippage limits, transaction construction | SDK helpers with current on-chain pool state |
| Indexer correctness checks | API plus direct account reads from `cp-amm` |
| Generated endpoint types | The OpenAPI section below |

For transaction-critical flows, read the pool account and quote through the SDK immediately before building the transaction.
